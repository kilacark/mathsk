Absolútna hodnota reálneho čísla

Nech a∈Ra\in {\bf R}a∈R. Absolútnu hodnotu čísla aaa označujeme ∣a∣\vert a\vert ∣a∣ a rozumieme ňou číslo

∣a∣=a, ak a≥0;\vert a\vert =a, \hbox{ ak } a \geq 0;∣a∣=a, ak a≥0;

∣a∣=−a, ak a<0.\vert a\vert = -a, \hbox{ ak } a<0. ∣a∣=−a, ak a<0.

Nech a,ba,ba,b sú reálne čísla. Potom platí:

  3. a≤∣a∣a \leq \vert a\vert a≤∣a∣

  5. ∣ab∣=∣a∣∣b∣\vert ab\vert = \vert a\vert \vert b\vert∣ab∣=∣a∣∣b∣
  6. ∣an∣=∣a∣n\vert a^n\vert = \vert a\vert^n ∣an∣=∣a∣n, pre každé prirodzené číslo nnn
  7. ∣ab∣=∣a∣∣b∣  (b≠0)\vert \frac{a}{b}\vert =\frac{\vert a\vert}{\vert b\vert}\ \ ( b \neq 0)∣ba​∣=∣b∣∣a∣​  (b=0)
